# 银龄智护适老化客户端

## 项目介绍
银龄智护适老化客户端是一款专注于为老年人提供智能护理服务的前端应用，旨在通过科技手段提升老年人的生活质量。

本项目前端页面基于HTML、CSS和JavaScript开发，具备完善的适老化功能，如字体大小调整、高对比度模式、语音朗读等，方便老年人使用。

## 软件架构
本项目采用前端经典的HTML+CSS+JavaScript架构，主要特点如下：
- 页面结构清晰，采用语义化HTML标签
- 样式统一管理，使用CSS实现响应式布局
- 交互逻辑通过JavaScript实现，包括用户认证、信息展示等功能
- 引入Font Awesome图标库增强视觉体验
- 使用Axios进行网络请求处理

## 功能模块
1. **用户中心**：用户登录、注册、信息管理等功能
2. **首页**：展示服务概览、个人信息等内容
3. **服务查询**：提供养老相关服务的查询功能
4. **社区反馈**：允许用户提交反馈意见
5. **在线办理**：支持养老相关业务的在线办理
6. **养老地图**：展示周边养老服务设施
7. **联系我们**：提供联系方式和沟通渠道

## 适老化特性
1. **字体调整**：支持字体大小增大、减小和重置
2. **高对比度模式**：提供高对比度显示选项，方便视力不佳用户
3. **语音朗读**：关键操作和信息支持语音朗读功能
4. **简洁界面**：页面设计简洁明了，减少操作复杂度
5. **响应式布局**：适配不同设备屏幕，包括手机、平板等

## 安装教程
1. 克隆本仓库到本地
2. 直接通过浏览器打开index.html文件即可运行

## 使用说明
1. 打开应用后，可通过顶部导航栏访问各个功能模块
2. 首次使用需注册账号，选择用户类型（老年人、家属或管理员）
4. 利用辅助工具栏可进行字体调整、对比度切换等操作
5. 可通过搜索功能快速查找所需服务
